# VIO_Stereo_MSCKF
A state estimator for robots using ESKF, which is based on msckf_vio/openvins stereo.

# 开发计划
编写一个应用于单个IMU与多目视觉融合的定位代码，以 IMU 的更新频率来发布最新状态估计结果。
+ 完成 data loader，控制多种不同传感器的数据输入和初步处理
+ 完成 imu propagate 和序列化过程
+ 完成加入多目视觉观测的 VIO 过程
